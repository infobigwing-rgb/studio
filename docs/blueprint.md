# **App Name**: Remian Edit Studio

## Core Features:

- Template Import: Import templates from Premiere Pro, After Effects, Final Cut Pro, and Apple Motion.
- Template Conversion: Convert imported templates into a web-compatible format.
- Layer Customization: Customize template layers with a layer-based property panel.
- Real-time Preview: Provide a real-time preview of the customized template using WebGL rendering.
- Format Export: Export customized templates to multiple formats, matching original template quality.
- Asset recommendations: AI powered recommendations of fonts and stock video and images
- Collaboration: Enable real-time collaboration with multiple editors on the same template project.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to reflect professionalism and creativity.
- Background color: Light gray (#F0F0F0), close in hue to the primary but highly desaturated, for a clean workspace.
- Accent color: Teal (#009688) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif for its neutral and modern appearance.
- Code font: 'Source Code Pro' for any code snippets used in tutorials or instructions.
- Use clean, professional icons to represent different tools and features.
- Editor interface layout: Left - Template Library, Center - Canvas, Right - Property Inspector, Bottom - Timeline.
